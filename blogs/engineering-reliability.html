Article 2: Technical Architecture

The Architecture: A Modern MERN Approach

When building a system meant for emergencies, the two most important technical requirements are Speed and Availability. A responder scanning a code shouldn't wait 10 seconds for a page to load.

To achieve this, I chose a decoupled architecture using the MERN Stack, deploying the frontend on Netlify and the backend on Render to ensure high availability.

Frontend: Beyond the Basics with React 18 & Vite

While many developers still use Create React App (CRA), I opted for Vite for superior performance and optimized builds.

  • Performance: Vite uses native ES modules for instant server starts.
  • Build Optimization: Viteâ€™s Rollup-based build ensures a minimal production footprint.
  • HMR: Near-instant reflection of UI changes during development.

UI Strategy: Tailwind CSS
Tailwind allowed for a responsive, high-contrast interface critical for emergency visibility.
